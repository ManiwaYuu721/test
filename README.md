StylePort ModellingHelper
===========

Roovモデリング効率化スクリプト。
[Roovモデリング仕様書](https://docs.google.com/document/d/1OZxZXrfExG3F3qZspyUPLhEylq62fEOCp8XB-6Zzq_I/edit#heading=h.tf3wwhaqq77)に記載されているフォーマットに準じた作りになっているか確認したり、
フォーマット通りにオブジェクトを自動で書き換えたりするツール(プラグイン)群

## Description
* [ ] 参照資料： https://styleport.esa.io/#path=%2F%E2%98%85%20Modelling

## Install
* Maya : Windows 10
	* 環境変数にMAYA_PLUG_IN_PATHを登録しMayaプラグインが格納されているディレクトリを登録、<br>Maya->ウィンドウ->設定/プリファレンス->プラグインマネージャ->作成したプラグイン選択->ロード<br>
